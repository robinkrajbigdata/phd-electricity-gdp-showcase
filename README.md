# Electricity Consumption and GDP Forecasting

This repository presents a simplified showcase version of my PhD-related research on forecasting macroeconomic activity using electricity consumption data.

## Research focus
The project examines whether electricity consumption patterns can help predict GDP dynamics using econometric and machine learning methods.

## Methods used
- Data preprocessing and feature engineering
- ARIMA and VAR models
- LSTM neural networks
- Forecast evaluation using metrics such as MAPE and R²

## Repository note
This is a public showcase version created for portfolio purposes. Full code, full dataset, and thesis-specific implementation details are not included.

## Tools
Python, pandas, numpy, matplotlib, statsmodels, tensorflow

## Example workflow
1. Load and preprocess electricity consumption data
2. Create forecasting features
3. Train a forecasting model
4. Evaluate prediction performance
5. Visualize results
